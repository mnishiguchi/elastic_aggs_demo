#### Example data models

## Get started

```
bin/setup
```

```
rake db:seed:import_noaa_stations          # Import NOAA weather station data
rake db:seed:import_noaa_weather           # Import NOAA weather CSV
```

```
npm install
```

```
elasticsearch
```

```
rails s
```
